'' QuizWeb - Interactive Quiz Application''
Quiz web application with admin login, admin panel, add/delete questions in admin panel, timer functionality

## Features

- ✅ User Quiz Interface
  - 30-second timer per question
  - Auto-advance when time runs out
  - Instant feedback (correct/wrong)
  - Progress bar
  - Final score with statistics

- ✅ Admin Panel (CMS)
  - Secure login system
  - Add unlimited questions
  - Set marks for correct answers
  - Set negative marks for wrong answers
  - Delete questions
  - View all questions

- ✅ Scoring System
  - Positive marking for correct answers
  - Negative marking for wrong answers
  - Zero marks for unanswered questions
